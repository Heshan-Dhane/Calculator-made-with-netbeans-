Simple Java CalculatorA functional desktop calculator application built with Java Swing and designed using the NetBeans IDE. This project was developed as part of my software engineering studies to practice GUI design and event handling. 🚀 FeaturesStandard Arithmetic: Supports addition, subtraction, multiplication, and division. Advanced Functions: Includes square root ($\sqrt{x}$), square ($x^2$), reciprocal ($1/x$), and percentage calculation. Modern UI: Integrated with the FlatLaf (Flat IntelliJ Look and Feel) for a sleek, clean interface. Error Handling: Built-in logic to prevent division by zero errors. 🛠️ Built WithLanguage: Java Framework: Swing (JFrame, GridBagLayout) External Library: FlatLaf for modern UI styling. IDE: NetBeans
